# TodoAssist - 제품 요구사항 문서 (PRD)

> AI 업무 비서 데스크톱 앱 — 할 일을 체계화하고 미루기를 방지하는 스마트 스케줄러

---

## 1. 프로젝트 개요

### 프로젝트명
**TodoAssist** — AI 업무 비서 데스크톱 앱

### 비전 (Vision)
제조업 품질 엔지니어들이 갖가지 할 일들을 효율적으로 관리하고, AI의 따뜻한 격려와 함께 미루기를 극복하는 생산성 도구

### 미션 (Mission)
- 머릿속과 메모장에만 존재하는 할 일들을 구조화된 시스템으로 전환
- AI가 지능적으로 30분 타임박스를 제안하고 강제
- 상황별로 적응하는 격려 메시지로 동기부여
- 통계 분석으로 업무 패턴을 파악하고 개선

## 3. 핵심 결핍 분석

### 결핍 1: 구조화된 업무 관리 시스템 부재
- **증상**: 할 일이 메모장, 카톡, 이메일에 분산됨
- **영향**: 중요한 태스크 누락, 우선순위 혼란, 업무 흐름 비효율
- **해결책**: 중앙화된 태스크 관리 시스템 구축

### 결핍 2: 미루기 습관 극복 불가
- **증상**: "나중에 하자" 반복 → 마감 며칠 전 급하게 처리
- **영향**: 업무 품질 저하, 스트레스 증가, 밤샘 작업
- **해결책**: AI 넛지와 30분 타임박스 강제로 지속적 진행

### 결핍 3: 업무 분해 능력 부족
- **증상**: 큰 업무를 바로 시작하거나, 너무 미세하게 쪼갬
- **영향**: 진행 상황 파악 어려움, 동기 유지 곤란
- **해결책**: 계층형 태스크 트리 + 자동 진행률 계산

### 결핍 4: 개인 업무 패턴 이해 부재
- **증상**: 어디서 시간을 쓰는지 모름, 반복되는 미루기 원인 불명
- **영향**: 개선 포인트를 모르고 같은 실수 반복
- **해결책**: 통계·분석·AI 인사이트 제공

## 4. 핵심 기능 요구사항

### F1: 30분 타임박스 스케줄링
**목표**: AI가 지능적으로 타임박스를 제안하고 사용자가 유연하게 조정

- AI가 태스크 난이도·데드라인·예상시간을 고려해 초안 스케줄 자동 생성
- 칸반 보드 형태로 30분 슬롯(0~47, 총 24시간) 시각화
- 드래그앤드롭으로 타임박스 위치 자유롭게 조정 가능
- 현재 진행 중인 태스크 하이라이트
- 일일 스케줄 저장 및 반복 사용 가능 (템플릿화)

### F2: 태스크 트리 + 진행률
**목표**: 복잡한 업무를 계층형으로 분해하고 자동으로 진행률 추적

- 무한 중첩 트리 구조 (예: 품목{승인서류{a, b, c}, 검사{x, y}})
- 리프 노드: 사용자가 수동 입력
- 부모 노드: 자식의 진행률 가중 평균 자동 계산
- 필터 기능: 카테고리, 중요도, 데드라인 기준 필터링
- 템플릿 저장/불러오기: 반복되는 업무 구조 재사용 (예: "월간 품질 보고서", "신제품 검수 체크리스트")
- 드래그앤드롭으로 태스크 재정렬

### F3: AI 격려 비서
**목표**: 상황에 맞는 톤으로 사용자를 동기부여

- **상황별 격려**:
  - 아침 인사: 따뜻하고 긍정적 톤
  - 태스크 시작: 격려와 함께 집중력 강조
  - 태스크 완료: 축하와 다음 스텝 제시
  - 미루기 감지: 긴박하지만 따뜻한 톤으로 넛지
  - 마감 임박: 긴급 알림 및 빠른 행동 권유
- 메시지는 Claude API로 생성, 사용자 프로필 기반 개인화
- 모든 상황(시작, 완료, 미루기, 마일스톤)에서 자동 발생

### F4: 통계/리포트
**목표**: 과거 데이터로 패턴을 분석하고 개선점 제시

- **주간 리포트**: 완료율(%), 예정 vs 실제 완료 비교
- **월간 리포트**: 장기 추세, 카테고리별 시간 분포
- **미루기 패턴 분석**: 어떤 유형의 태스크를 미루는가? 언제 미루는가?
- **카테고리별 시간 분포**: 원형 차트, 예상 시간 vs 실제 소요 시간
- **AI 인사이트**: "일반적으로 30분을 60분 소요하십니다", "금요일에 미루기가 2배 높습니다" 등
- 차트 형식: 선 그래프(추세), 막대 그래프(비교), 원형 차트(분포)

## 5. 비기능 요구사항

### 5.1 성능
- **앱 시작 시간**: 2초 이내
- **화면 전환**: 300ms 이내
- **타임박스 드래그**: 60fps 유지
- **데이터베이스 조회**: 100ms 이내
- **로컬 저장소 크기**: 최대 1000개 태스크 기준 <50MB

### 5.2 보안
- **로컬 데이터**: SQLite 파일 기반, 암호화 옵션 제공
- **Claude API**: API 키를 환경 변수로 관리, 로컬에만 저장
- **개인정보**: 모든 데이터는 로컬 저장, 클라우드 동기화 없음
- **권한**: 특정 폴더 접근권만 요청

### 5.3 사용성
- **아이콘/텍스트**: 명확한 UI, 초보자도 쉽게 이해 가능
- **온보딩**: 첫 실행 시 3분 내 기본 사용법 설명
- **접근성**: 휠체어 사용자도 사용 가능하게 Tab/Enter 네비게이션 지원
- **기본 언어**: 한국어

### 5.4 신뢰성
- **로컬 백업**: 앱 종료 전 자동 백업
- **오프라인 작동**: 인터넷 없이도 기본 기능 작동 (API 호출 제외)
- **에러 복구**: 비정상 종료 후 데이터 무결성 보장

### 5.5 유지보수성
- **로그**: DEBUG/INFO/ERROR 레벨 로깅, 로그 파일 저장
- **버전 관리**: 데이터 마이그레이션 스크립트 자동 실행
- **테스트**: 핵심 로직 유닛 테스트 커버율 80% 이상
